# to_do_list
This project is a basic task-management tool built using Java. It helps users keep track of their daily activities in an organised way.
# Todo List Manager

A simple command-line todo list application built with Java.

## Features
- Add new todo items
- Mark items as completed
- Remove items
- View all items with creation timestamps
- Persistent menu interface

## Requirements
- Java 11 or higher
- Maven 3.6 or higher

## Building and Running

### Compile and Run:
```bash
mvn compile
mvn exec:java -Dexec.mainClass="com.todo.Main"
